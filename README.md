[iFound](https://github.com/antonio-emilio/Estrutrura_de_dados1) é um Software de organização de estoque, desenvolvido especialmente para acompanhar localização de componentes eletrônicos. Com a popularização de SMD's e PTH's cada vez menores e de difícil identificação, com o sistema iFound você poderá catalogar seus componentes, separando-os por categoria, quantidade, especificação etc.


### Documentação
Com o iFound, você poderá:
* Cadastrar categorias;
* Cadastrar componentes;
* Procurar componentes;
* Obter a localização de cada componente cadastrado;

E muito mais!
